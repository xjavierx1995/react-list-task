# React App (react-list-task)

This project was made with react, redux, chakra-ui and typescript

## Demo

[text](https://preeminent-scone-c8b0bb.netlify.app/)

## Install the dependencies

```bash
npm install
```

### Start app

```bash
npm run dev
```

### Execute tests

```bash
npm run test
```

### Execute tests coverage

```bash
npm run coverage
```

```bash
npm run coverage:ui
```

### Dependencies

This project use:

- Node 20.10.0
